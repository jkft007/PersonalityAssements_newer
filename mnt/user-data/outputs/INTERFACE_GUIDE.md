## ğŸ¨ Color/Style Reference

### Text Input Fields
- **Background**: Light gray/white
- **Border**: Thin gray border
- **Text**: Dark gray/black
- **Placeholder**: Light gray italic
- **Info text**: Small gray text below field

### Buttons
- **Export buttons**: Large, blue/primary color
- **Clear button**: Gray/secondary color
- **Hover effect**: Slightly darker shade

### File Downloads

- **After export**: Clickable download link with filename

## ğŸ’¡ Tips for Users

### Visual Cues
1. **Pre-filled text**: Shows suggested default name
2. **Cursor in field**: Ready for editing
3. **Info text**: Explains automatic extension handling
4. **File output area**: Shows download when ready

### Editing Tips
1. Click in the text field
2. Select all (Ctrl+A / Cmd+A) or click-drag
3. Type new name
4. Click export button
5. File downloads automatically

### What You See vs What You Get
